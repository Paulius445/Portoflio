Pegasas

Description

It was my first project during my internship. It was based on redesigning the online store for a real publishing house.

Tasks

  • Design a mega menu.
  • Design a category page.
    • All these tasks had to be done while working with an existing design system.

    Process

    1. Analyzing the tasks and given data (research data, documentation) to clarify goals for every page I was designing.

    2. Wireframing. Wireframing helped me get feedback for the further steps of high fidelity designs.

    3. Designing hi-fi screens and interfaces. Making sure the product is aesthetically pleasing and functional at the same time.

    4. Testing/QA. Getting feedback and improving on my work. Also comparing my designs with real solutions of this project.

    Results and lessons

    This project provided a valuable learning experience that allowed me to gain hands-on experience in the field of interface design and user experience for eCommerce projects. I also got some experience working with real design system.
